# Bundled font — DejaVu Sans

`DejaVuSans.ttf` is DejaVu Sans 2.37, bundled here so the Lambda runtime (which
has no system fonts) can render the sealed Future Predictions keepsake PNG.

DejaVu fonts are free to use and redistribute under a permissive license (the
Bitstream Vera license plus the Arev/DejaVu additions). Full text:
https://dejavu-fonts.github.io/License.html

Source of this copy: the `dejavu-fonts-ttf@2.37.3` npm package
(https://cdn.jsdelivr.net/npm/dejavu-fonts-ttf@2.37.3/ttf/DejaVuSans.ttf).
